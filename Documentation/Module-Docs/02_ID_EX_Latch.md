## Functionality
### Registers
  - 5-bit rdn register
  - 32-bit a register
  - 32-bit b register
  - 32-bit branch_addr register
  - 32-bit pc register
  - 32-bit rs2d register
  - 1-bit branch_taken register
### On posedge clk
  - ```rdn = rdn_in```
  - ```branch_taken = branch_taken_in```
  - ```rs2d = rs2d_in```
  - ```branch_addr = branch_addr_in```
  - ```pc = pc_in```
  - ```a_sel```

    |State|Output|
    |---|---|
    |```a_sel == 0```|```a = rs1d```|
    |```a_sel == 1```|```a = pc```|
    |```a_sel == 2```|```a = 0```|
  - ```b_sel```

    |State|Output|
    |---|---|
    |```b_sel == 0```|```b = rs2d_in```|
    |```b_sel == 1```|```b = imm```|
    |```b_sel == 2```|```b = 4```|
    |```b_sel == 3```|```b = imm << 12```|

### Asynchronous active low reset
  - Register values reset to 0
